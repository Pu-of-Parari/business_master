# Business_master
[COTOHA API](https://api.ce-cotoha.com/contents/index.html)を用いた自動敬語変換ツール

絶賛開発中

## 目的
- 一般的な言葉遣いの文を敬語表現を用いた文に自動校正するツールを作る

- 裏目的
  - フロントエンド修行
  - Git開発に慣れる
  - agileっぽく開発できたらいいな

## 仕様
`input: str` 一般的な言葉遣いの文章

`output: str` ビジネス敬語に変換された文書

フロント、サーバに関する仕様も順次追加していく
